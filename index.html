Features that keep our users happy

Full control of script execution flow

PyWebIO doesn't automatically re-execute the whole script for any input changes. It supports synchronization, callback and coroutine to obtain and process user inputs. With PyWebIO, there will be no more confusion in "app states" and "caching issues". 

Balance between low code and flexibility

Existing Python script can be transformed into a Web application only by modifying the input and output operations using PyWebIO functions. You also gain flexibility by using almost any Python libraries that you're already familiar with.

Working with web services of your choice

Integrating PyWeb apps with your existing web services is easy. We currently support many popular web frameworks, such as Flask, Django, Tornado, aiohttp, FastAPI. It's also straigthforward to integrate 3rd-party web services (e.g. authentication, cloud API).
