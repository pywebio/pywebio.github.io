Upload Assets‚Äã

From the directory containing the index.mdx file, click the "Add file" button and select "Upload files". You can upload as many files as needed from this screen. After you are finished, ensure the "Commit directly to the blog-date-description branch" option is selected and click the "Commit changes" button.

You will be re-directed to the home screen of the branch you just committed to, navigate back to the index.mdx file in the blog or docs(tutorials) directory to continue editing. To add the uploaded assets to your document, use the following relative file path:

  • ./your_asset.jpg

Add an image by using the exclamation mark(!), followed by alt text in brackets[], and the relative path or URL to the image in parenthesis(). A title can be added in quotation marks("") as well after the relative path or URL.

To insert tabs, you must first import the tab components in your file (directly under the front matter) and then wrap your content in the <Tabs> component. For more information on customizing tabs, check out this documentation.

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';
<Tabs>
  <TabItem value="baseball" label="Baseball" default>
    <b>This is a baseball</b> ‚öæ
  </TabItem>

    Insert Custom Button‚Äã

    Custom Buttons can be created with the use of JSX. Styling a button is very similar to CSS and you can pass in as many arguments as needed to avoid multiple functions for multiple buttons.

    note

    Styling properties belong in between the double style={{ }}. {children} is a necessary parameter and must be referenced outside of the style properties as shown below.

    Examples:

    export const Button = ({children, backgroundColor, fontColor, borderStyle, borderColor}) => (
      <span
        style={{
          backgroundColor: backgroundColor,
          borderRadius: '.313rem',
          borderWidth: 1,
          borderStyle: borderStyle,
          borderColor: borderColor,
          color: fontColor,
          fontFamily: 'Segoe UI',
          fontSize: '0.8rem',
          fontWeight: '600',
          paddingTop: '0.3rem',
          paddingBottom: '0.35rem',
          paddingLeft: '1.0rem',
          paddingRight: '1.0rem',
        }}>
        {children}
      </span>    
    )

    Button with default black font

    <Button backgroundColor='#0f9d58'>Button</Button>

    Button with white font

    <Button backgroundColor='#0f9d58' fontColor='white'>Button</Button>

    Button with a border

    <Button backgroundColor='white' borderStyle='solid' borderColor='black'>Button</Button>
    note

    Dark mode will change default black text to white. Dark mode will not change the font color if specified.
